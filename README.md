# psarlarn_INFO250
## Project Overview
PsarLarn is a website that was built using HTML 5 and CSS 3. Psarlarn is an automotive dealership website that aims to assist Cambodian consumers in the navigation and exploration of vehicles. 
## Objective
- Provide an easy way for users to browse available cars
- Display detailed vehicle information and specifications
- Allow users to contact dealers directly
- Create a clean, responsive, and user-friendly interface
## Website Structure 
Psarlarn consists of 4 main pages:
## Home Page
- Introduction banner (poster-style car image)
- Tagline and “Browse Cars” button
- Featured / limited car models
- Key stats and benefits
- Footer with contact & social links
## About Page
- Company history
- Core values
- Team members
- Customer feedback
## Cars Page
- Grid layout of car listings
- Price and basic info
- “View Details” button
### Car Details Page
- Multiple car images
- Specifications table
- Description
- Contact buttons (Facebook, Telegram, Email)
## Contact Page
- Contact form
- Dealership information
- Embedded Google Map
- Social media links
## Languages Used 
- HTML5 – structure and semantic layout
- CSS3 – styling and design
